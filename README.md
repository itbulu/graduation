周末的时候练手Typecho主题模板，同样前端样式寻找在网上的免费开源的框架，然后修改整合到Typecho适用的主题。基于毕业季，就姑且叫做graduation主题，好歹有个名字就这么取名。

**主题特点：**
1、简单的双栏主题，适合个人博客
2、自动缩略图随机，如果需要自定义缩略图需要在附件中添加图片
